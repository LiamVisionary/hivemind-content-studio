Create a spot-the-AI split-screen post: one side is a real creator clip, the other is an AI render built to be indistinguishable.

Beats:
1. [Split screen] left: [REAL CLIP]. Right: an AI-generated clip of [AI CLIP SUBJECT] built with the full UGC realism system — locked character reference, named imperfections, lived-in environment, phone-camera language, complete negative stack. Both sides same framing, same energy, same duration.
2. [The ask] a single on-screen or caption question: "one of these is a real ugc creator, the other is fully ai. which is which?"
3. [Aftermath] answer every reply; replies are the heaviest engagement signal and the answering keeps the thread alive.

Rules:
- No product, no CTA, no brand mention in the frame — reach is the point of this post; conversion posts come later.
- The AI side must genuinely hold up; if it's clockable in one glance, improve the realism prompt before posting.
- This format doubles as a public quality benchmark for your AI UGC pipeline.
