Create a mascot-engine video for [PRODUCT] starring [MASCOT — species/design, personality, one visual signature].

Beats:
1. [MASCOT] shows up somewhere it plausibly shouldn't be and pursues today's side quest: [SIDE QUEST] (a mood, an errand, a petty feud — never a product pitch).
2. The mascot has a consistent personality and one running bit that carries across every episode.
3. The product appears only as set dressing or context; the mascot never presents it.

Rules:
- The mascot lives a life; it does not sell. Recognition is the mechanism — design for being identified in 0.2 seconds (same silhouette, same palette, same energy every post).
- Keep a reusable character reference so the mascot is identical across episodes; this template is built for a series, not a one-off.
- Unhinged is allowed and encouraged; off-brand for a corporate account is exactly on-brand for the mascot.
