Create a batch brief for [PRODUCT] in the trend-template-volume format: clone this week's proven trend as a repeatable template and produce [VARIATION COUNT] variations.

Beats per variation:
1. Recreate the current trend [TREND — the dance, sound, or format going viral this week] with the product's output or usage slotted into the trend's variable slot.
2. Keep the trend's bones identical — timing, structure, sound — and vary only the surface: different character reference, different room, different opening line.
3. Caption casual and first-person, different per variation.

Rules:
- Don't invent, reprint: the format was already validated by the algorithm. Find a clip already going viral in the niche and deconstruct it beat by beat before generating.
- Volume is the strategy: kill losers weekly, feed winners. No single variation needs to be special.
- Use one locked character reference per creator identity so variations batch cleanly; the realism system applies to every render.
